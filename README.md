## Usage

### Testing

Run tests

```
mvn test
```

Generate test coverage report

```
mvn jacoco:report
```

### Other

Generate JAR

```
mvn package
```

Generate JavaDoc

```
mvn javadoc:javadoc
```

Use checkstyle

```
mvn jxr:jxr checkstyle:checkstyle
```
